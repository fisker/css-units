# css-units

> List of CSS:Cascading Style Sheets Units

Data from [CSS Values and Units Module Level 4](https://www.w3.org/TR/css-values-4/)

## Install

```bash
yarn add css-units

# OR with npm

npm install --save css-units
```

## Usage

```js
import cssUnits from 'css-units'

console.log(cssUnits)

// => ['em', 'ex', 'cap', 'ch', ...]
```
